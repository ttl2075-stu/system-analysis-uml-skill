# system-analysis-uml-skill

Design-first **System Analysis & Design** skill pack for AI agents, with UML intent and Mermaid-first notation.

## What this skill provides
- Method core (8-step workflow): requirements → analysis → design → traceability
- Notation rules for UML/Mermaid diagrams
- Runtime guardrails (evidence-first, no step-skipping)
- Templates for docs and diagrams
- Validation and quality artifacts

## Primary entry
- `SKILL.md` (main skill instructions)

## Quick use
1. Read `SKILL.md`
2. Follow `method-core.md` step by step
3. Apply `notation-rules.md` and `diagram-qa-checklist.md`
4. Use templates in `templates/` and `templates/mermaid/`
